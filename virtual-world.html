Therapy Virtual World

(Nov 2023 - Jun 2024)
  • Project Type: group of ~5 assisting a grad student's research
  • Software: Unity, Visual Studio
  • Other Tools: Fishnet, VR
  • Language: C#

This project aimed to evaluate the potential of virtual worlds as a platform to conduct virtual counseling. The virtual world supports multiplayer, voice and text communication, emotes, character customization, and building structures.

This project was led by a graduate student working on his thesis. I and 3-5 other undergraduate students were recruited to assist him.

My Contributions:
  • UI : refined character customization menus, created help menu and action bar
  • Graphics Optimization : implemented LOD in set dressing to optimize performance
  • Building Mechanics : implemented the building ability. The player can spawn various shapes and objects, edit their size, shape, and position.
  • Multiplayer Networking : ensured changes to the player's appearance, chat messages, and spawned objects synchronized with all other players on the server
