Components :

  • Dialogue System : designed and programmed a dialogue system which supports branching dialogue, event triggers, and other customizations
  • Gameplay Mechanics : 4 minigames with completely different gameplay and controls. Currently these include a box organization puzzle game, and a cashier item scanning challenge
